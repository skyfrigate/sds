==========
User Guide
==========

Welcome to the SDS-Tools user guide! This guide provides comprehensive tutorials
and explanations for using the library effectively.

About This Guide
================

This user guide is organized into several sections, each covering a family of
data structures. Each section includes:

* **Theoretical foundations** - Mathematical models and formal definitions
* **Algorithmic analysis** - Implementation strategies and complexity analysis
* **Practical usage** - Code examples and real-world applications
* **Best practices** - Tips, patterns, and common pitfalls

.. note::

   This guide focuses on concepts and usage. For detailed API documentation,
   see the :doc:`../api/index`.
