Deprecation Policy
==================

Features marked as deprecated will:

1. Be clearly marked in documentation
2. Emit deprecation warnings when used
3. Remain functional for at least 2 minor versions
4. Be removed in the next major version

Upgrade Guide
=============

When upgrading between versions, check the following:

From 0.0.x to 0.1.0
-------------------

* No breaking changes
* New features added
* Performance improvements
